## 🚀 Featured Project

### 🤖 Personal AI Assistant (RAG Chatbot)

A **context-aware AI chatbot** that can answer questions about me by retrieving information from a custom knowledge base.

#### 🔥 Key Features:
- 🧠 RAG-based architecture (Retrieval + Generation)
- 💬 Conversation memory (chat history aware)
- 📄 Custom knowledge base using personal data
- 🎯 Third-person professional responses
- ⚡ Real-time response streaming
- 🌐 Deployed using Streamlit Cloud

#### ⚙️ Tech Stack:
- Python
- LangChain
- ChromaDB (Vector Database)
- HuggingFace Embeddings
- Groq API (LLM)
- Streamlit
